# adaptive-oscillator-ocr

# Associative Learning in Adaptive Oscillator Networks

## Overview
This project studies pattern recognition using networks of coupled phase oscillators.
We implement the Kuramoto model and investigate synchronization-based memory.

## Stage 1
- Task 0: Kuramoto model for N=2 oscillators
- Task 1: Kuramoto model for N>2 oscillators

## Goals
- Understand synchronization
- Implement numerical ODE solvers
- Study phase locking

## Tools Used
- Python
- NumPy
- SciPy
- Matplotlib
